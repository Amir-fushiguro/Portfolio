Solo Indie Game Developer · Berlin, Germany

Hey there! I'm Alex — a solo indie developer who's been building games professionally since 2019 (and making terrible little prototypes since I was 12).

I specialize in atmospheric games with deep mechanical systems. Each project starts with one weird mechanic I can't stop thinking about — then I build a whole world around it.

I handle everything: design, code, art direction, and soundtrack. Currently building an unannounced open-world survival game in a snowy world — naturally! ❄️
